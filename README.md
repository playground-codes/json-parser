### Just a test JSON Parser in .NET 

**IDE**: Visual Studio Community  
**Language**: C#  
**Framework**: .NET Core 2.0  

### Play with it

#### Run Console program
```
dotnet run
```  
#### Restore dependences and tools
```
dotnet restore JSONParser.csproj
```
#### Run unit tests
Using [xUnit.net](https://xunit.github.io/)  
```
dotnet xunit
```
